# SDG 17: Partnerships for the Goals

This project is a single-page website themed around **UN Sustainable Development Goal 17 (SDG 17)**.  
It highlights the importance of global partnerships in achieving sustainable development.

---

## Features

- Hero section introducing SDG 17  
- About section with key SDG 17 targets  
- Partnership logo carousel  
- Responsive navigation menu  
- Social media footer
